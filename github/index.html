github

GitHub是一个软件项目的托管平台，其只支持Git作为版本库。它还是一个开源协作社区，通过GitHub，既可以让别人参与你的开源项目，也可以参与别人的开源项目.

注册

1 打开https://github.com,点击注册按钮

2 填写注册信息

修改信息

注册完毕登陆成功之后。选择settings进行基本资料的一些设置

代码查找

点击回到首页

输入搜索内容，显示结果列表

进入代码页 watch：关注当前项目的动态，会给你的邮箱发邮件 star：感觉项目好，给个赞 fork：把项目变成自己的，随意修改
